# 余额汇总 & 消耗处理 自动化脚本使用文档

# 注意：command 是 mac系统、bat 是 windows 系统，下面文档结构示例为 mac 图标

同时系统需要安装 Python3.9+版本以上

## 目录结构

```
py 自动化脚本/
├── 一键运行-创建日期目录.command    ← 双击创建当天目录
├── 一键运行-全部消耗.command       ← 双击处理所有消耗数据
├── 一键运行-余额汇总.command       ← 双击处理余额数据
├── Python/                        ← 脚本代码目录
│   ├── 创建日期目录.py
│   ├── 消耗汇总/                   ← 消耗处理脚本 + 回填模板
│   └── 余额汇总/                   ← 余额处理脚本
├── 2026-07-08/                    ← 日期目录（自动创建）
│   ├── 原始文件/                   ← 放入原始数据
│   ├── 生成文件/                   ← 脚本输出结果
│   └── 失败文件与日志/             ← 错误日志
└── ...
```

---

## 使用流程

### 第一步：创建日期目录

双击 `一键运行-创建日期目录.command`

自动创建当天的目录结构，包含所有平台的原始文件和生成文件文件夹。

### 第二步：放入原始文件

将各平台下载的报表文件放到对应的 `原始文件` 子目录中：

| 目录 | 放什么文件 |
|------|-----------|
| 磁力- 余额汇总/ | 快手磁力导出的ZIP压缩包 |

- **处理余额**：双击 `一键运行-余额汇总.command`
- **处理消耗**：双击 `一键运行-全部消耗.command`

每个平台目录如果为空或不存在会自动跳过，不影响其他平台。

## 处理指定日期

所有脚本默认处理当天日期。如需处理其他日期，在终端中运行：

```bash
cd Python/消耗汇总
python3 广点通ADQ_消耗处理.py 2026-07-08
python3 快手消耗处理.py 2026-07-08
python3 总消耗汇总.py 2026-07-08
```
